# 키보드 & 터치

> 화면 키보드, numpad, 날짜 선택기 및 기타 터치 컨트롤 사용법입니다.

---

## 화면 키보드

시스템은 터치스크린 사용을 위해 설계되었습니다. 일상 운영에 물리 키보드가 필요하지 않습니다.

### 키보드 레이아웃

| 레이아웃 | 용도 |
|----------|------|
| **한국어** | 한국어 텍스트 입력 (매장명, 상품명, 메모) |
| **English** | 영어 텍스트 입력 (주소, 이메일, website) |
| **Numpad** | 숫자 입력 (코드, 가격, 전화번호) |

키보드는 필드에 따라 적절한 레이아웃을 자동으로 선택합니다. 예를 들어:
- Store name → 한국어 키보드
- Phone number → Numpad
- Email → English 키보드

### 인라인 키보드

일부 화면에서는 키보드가 페이지에 직접 표시됩니다 (예: Open Shift 메모, User Form). 필드를 눌러 선택하면 키보드가 해당 필드를 대상으로 합니다.

### 팝업 키보드 (KeyboardInputText)

검색 바와 간결한 필드는 **탭하여 열기** 키보드를 사용합니다:
1. 필드를 누르면 전체 화면 오버레이가 열립니다
2. 화면 키보드로 입력합니다
3. 키보드 밖을 누르거나 Enter를 눌러 닫습니다

현재 텍스트가 키보드 위의 큰 미리보기 말풍선에 표시됩니다.

## Cash Counter

Shift 시작 또는 마감 시 cash를 세는 데 사용됩니다:

| 구성요소 | 설명 |
|----------|------|
| 권종 그리드 | 지폐/동전 종류를 눌러 선택 |
| Numpad | 선택한 권종의 수량 입력 |
| Total | 수량 입력 시 자동 업데이트 |
| Kick Drawer | Cash 서랍 열기 |
| Reset | 두 번 누르기 리셋 (한 번 눌러 확인, 다시 눌러 실행) |

권종당 수량은 **999**개로 제한됩니다.

## Date Range Selector

날짜로 필터링할 때 사용됩니다 (invoice search, cash in/out):

1. 날짜 필드를 누르면 캘린더 오버레이가 열립니다
2. 나란히 두 개의 캘린더: **From** (왼쪽)과 **To** (오른쪽)
3. 날짜를 눌러 범위를 설정합니다
4. 프리셋 사용: **Today**, **This Week**, **This Month**, **This Year**
5. **Confirm**을 눌러 적용합니다
6. 날짜 필드를 다시 누르면 필터가 **해제**됩니다

## 페이지 목록

목록은 스크롤되지 않습니다 — 터치스크린 편의를 위해 **페이지 버튼**을 사용합니다:

| 버튼 | 동작 |
|------|------|
| ⬆⬆ | 첫 페이지 |
| ⬆ | 이전 페이지 / 이전 행 |
| ⬇ | 다음 페이지 / 다음 행 |
| ⬇⬇ | 마지막 페이지 |

두 가지 유형의 페이징:
- **Server paging** (ServerPagingList) — 위/아래 시 server에서 새 데이터를 가져옴
- **Client paging** (PagingRowList) — 모든 데이터가 로드된 상태, 페이징은 로컬 (refund 패널, sale line viewer에서 사용)

## 일반 터치 패턴

| 패턴 | 동작 |
|------|------|
| 누르기 | 선택, 활성화, 이동 |
| 이중 누르기 확인 | 중요 작업 (shift 마감, cart 비우기)에 두 번 누르기 필요 |
| 활성 필드 강조 | 현재 선택된 입력에 파란색 테두리 + 링 |
| 비활성 상태 | 회색 처리되고 투명도 감소 |
