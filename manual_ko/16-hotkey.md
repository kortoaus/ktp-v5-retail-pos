# Hotkey Manager

> Sale 화면에서 사용할 빠른 접근 상품 버튼을 설정합니다.

---

## Hotkey란?

Hotkey는 Sale 화면 오른쪽 (줄이 선택되지 않은 상태)에 표시되는 사전 설정된 상품 버튼입니다. 스캔 대신 한 번 누르기로 자주 판매하는 상품을 추가할 수 있습니다.

---

## 접근 방법

Home 화면에서 **Hotkey Manager**를 누릅니다. **hotkey** 권한이 필요합니다.

---

## 구조

Hotkey는 **그룹** (탭)으로 구성됩니다. 각 그룹에는 **6×6 그리드**의 상품 버튼이 있습니다.

| 개념 | 설명 |
|------|------|
| **Group** | 이름이 있는 탭 (예: "음료", "과자") + 색상 |
| **Cell** | 6×6 그리드의 하나의 위치 |
| **Key** | Cell에 할당된 상품 — 상품명과 그룹 색상 표시 |

---

## 그룹 관리

### 그룹 생성

1. **Add Group**을 누릅니다.
2. 화면 키보드로 이름을 입력합니다.
3. 색상을 선택합니다.
4. **Save**를 누릅니다.

### 그룹 편집

1. 그룹 탭을 선택합니다.
2. 그룹 이름/설정 영역을 누릅니다.
3. 이름이나 색상을 변경합니다.
4. **Save**를 누릅니다.

### 그룹 삭제

1. 그룹 탭을 선택합니다.
2. **Delete**를 누릅니다 — 확인이 필요합니다.

---

## 상품 할당

### Cell에 상품 추가

1. 그룹 탭을 선택합니다.
2. 그리드에서 빈 cell을 누릅니다.
3. 상품 검색 모달이 열립니다 — 상품을 찾아 선택합니다.
4. 해당 cell에 상품이 할당됩니다.

### Cell 편집

1. 상품이 있는 cell을 누릅니다.
2. 옵션: 표시 이름 변경, 색상 변경, 또는 cell 비우기.

### 상품 제거

1. 상품이 있는 cell을 누릅니다.
2. **Clear**를 눌러 해당 cell에서 상품을 제거합니다.

---

## 저장

변경 사항은 **Save**를 누를 때 그룹별로 저장됩니다. 저장하지 않은 변경 사항은 dirty 마커로 표시됩니다. 저장하지 않고 나가면 변경 사항이 취소됩니다.

---

## Sale 화면에서

Cart에서 줄이 선택되지 않은 상태에서 오른쪽에 hotkey 그리드가 표시됩니다. Hotkey 버튼을 누르면 해당 상품의 barcode를 스캔하는 것과 동일합니다 — 같은 가격 및 병합 로직을 사용하여 상품이 cart에 추가됩니다.

그룹은 탭으로 표시됩니다 — 탭을 눌러 hotkey 페이지를 전환합니다.
