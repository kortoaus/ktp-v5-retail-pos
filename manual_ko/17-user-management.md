# User Management

> 사용자 계정 생성, 편집, 관리 방법입니다.

---

이 페이지는 관리 인터페이스를 다룹니다. 사용자와 로그인 작동 방식은 [Terminal & Users](./02-terminal-and-users.md)를 참조하세요.

---

## 접근 방법

Home 화면에서 **User Management**를 누릅니다. **user** 권한이 필요합니다.

---

## 화면 구성

| 영역 | 설명 |
|------|------|
| 왼쪽 패널 | 검색 기능이 있는 사용자 목록 (페이지네이션) |
| 오른쪽 패널 | 사용자 양식 (생성 또는 편집) |

---

## 검색

검색 바에 keyword를 입력하고 **Search**를 누릅니다. 사용자 이름과 code를 검색합니다. 입력 중에 자동으로 검색되지 않습니다.

---

## 사용자 생성

1. 상단 바에서 **New**를 누릅니다.
2. 양식을 작성합니다:
   - **Name** — 한국어 이름은 한국어 키보드 사용
   - **Code** — 숫자만, numpad로 입력
   - **Scopes** — 권한 체크박스 선택
3. **Create**를 누릅니다.

Code는 활성 사용자 중에서 고유해야 합니다.

---

## 사용자 편집

1. 목록에서 사용자를 누릅니다.
2. 양식에 해당 사용자의 정보가 표시됩니다.
3. 변경합니다.
4. **Update**를 누릅니다.

---

## Archive

기존 사용자의 **Archived** 스위치를 켜고 **Update**를 누릅니다. Archive된 사용자는:
- 로그인할 수 없습니다
- 감사 목적으로 데이터베이스에 남아 있습니다
- 과거 sale과 작업이 보존됩니다

---

## 규칙

- **Admin 사용자** (ID 1)는 편집하거나 삭제할 수 없습니다
- 사용자 code는 비아카이브 사용자 중에서 고유해야 합니다
- 이미 존재하는 code로 사용자를 생성하려 하면 시스템이 거부합니다
- 사용자 목록에서 admin 사용자 (ID 1)는 제외됩니다
- 결과 정렬: 비아카이브가 먼저, 그 다음 ID순
